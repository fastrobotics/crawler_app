Setup

Scenario Management

To set the scenario, perform the following:

  1. Remove any config folders under catkin_ws/src/: ``bash rm -rf ~/catkin_ws/src/config/ ```
  2. Determine which scenario shall be used. Currently supported scenario's:
  • dev
  1. Create a symlink from this scenario to the config directory:
    cd ~/catkin_ws/src/
    ln ~/<location>/crawler_app/scenarios/<scenario>/ config
